STM32定时器:
	高级定时器(TIM1, TIM8)	
	通用定时器(TIM2, TIM3, TIM4, TIM5)
	基本定时器(TIM6, TIM7) 

/*-------------------------------------------------------------------------------------------------------*/

基本定时器(TIM6, TIM7)
	16位自动重装载累加计数器.
	16位可编程预分频.
	可触发DAC的同步电路.
	在更新事件(计数器溢出)时产生中断/DMA请求.

/*-------------------------------------------------------------------------------------------------------*/

通用定时器(TIM2, TIM3, TIM4, TIM5)
	16位向上, 向下, 向上/向下自动装载计数器.
	16位可编程预分频.
